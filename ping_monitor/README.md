# IP Ping 监控工具

这是一个实时监控IP地址连通性的工具，支持多IP监控、状态显示和报警功能。

## 主要功能

- IP地址管理（添加/删除/编辑/分组）
- 实时Ping检测
- 状态显示和可视化
- 多种报警方式

## 安装说明

1. 确保已安装Python 3.8或更高版本
2. 克隆此仓库
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行主程序：
   ```bash
   python src/main.py
   ```
2. 在图形界面中添加需要监控的IP地址
3. 配置监控参数和报警设置
4. 开始监控

## 配置说明

配置文件位于 `config/config.yaml`，可以修改以下设置：
- Ping间隔时间
- 超时阈值
- 报警阈值
- 报警方式

## 开发说明

项目结构：
- `src/` - 源代码目录
  - `main.py` - 主程序入口
  - `gui/` - 图形界面相关代码
  - `core/` - 核心功能模块
  - `utils/` - 工具函数
- `config/` - 配置文件目录
- `tests/` - 测试代码目录
